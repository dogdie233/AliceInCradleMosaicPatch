# AliceInCradleMosaicPatch
一个适用于游戏 _摇篮中的爱丽丝_ 的去马赛克补丁

__此补丁不受Alice In Cradle的官方支持__

# 安装和使用
1. 安装[BepInEx5](https://github.com/BepInEx/BepInEx/tree/v5-lts)（教程自己找）
2. 从[Release](https://github.com/dogdie233/AliceInCradleMosaicPatch/Release)下载适用于你的游戏版本的补丁（dll文件）
3. 将下载的dll放置在对应的插件文件夹

# 注意
只有在`健全模式`设置为 __关__ 时此插件才会生效

# 开发
拉取项目后需要在项目根目录新建一个`AICAssemblies`文件夹，并将`/游戏根目录/AliceInCradle_Data/Managed`中的所有dll放入(如果你会看csproj也可以只放入一部分)
